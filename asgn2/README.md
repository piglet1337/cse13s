Instructions to run Program:  
First build mathlib-test in the commandline with:  
make mathlib-test  
Then run it with:  
mathlib-test [arguments]  
ARGUMENTS:  
-a: to run all tests.  
-s: to run sin tests.  
-c: to run cos tests.  
-S: to run arcsin tests.  
-C: to run arccos tests.  
-T: to run arctan tests.  
-l: to run log tests.
